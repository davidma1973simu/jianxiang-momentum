## 1. 产品目标与成功指标

**产品目标**：成为用户"事一卡就摸"的极简向内求工具，用三层改变（心念→语言→行为）帮他扩容心量。

**北极星指标**：**周主动触发率**——用户在"大象"出现时主动打开见象的比例（不是 DAU、不是连胜）。

**成功指标（对比行业崩塌基线：中位 30 日留存 3.3%、DAU 4.0%）**
| 指标 | 目标 | 说明 |
|------|------|------|
| 30 日留存 | > 25% | 行业约 5%，靠无 guilt + 生活触发拉开 |
| 单次会话时长中位 | 5–60 秒 | 轻，不占时间 |
| phraseTouched 周触达率 | 验证语言层真正落心 | 「触动了我」轻触点比率 |
| 深度工具月活占比 | 验证"用户拉杠杆"假设 | 四步照象 / 过滤网使用率 |

**非目标（不做的方向）**：冥想计时、写作日记、诊断、诱导转化。

## 2. 核心用户流

### Flow A · 日常轻循环（默认路径，零强制）

```
[晨间锚点] 1 分钟，可跳过
  选其一：①关注感受 ②关注小事 ③关注关系
  → 下方淡显一句活人话提示（5 秒消失，非输入）

[大象暂停法] 随时触发（事一卡就摸见象）
  停：震动 + 息屏 3 秒（行为阻断）；同时在心里认一声"大象来了"——
      把"又卡住了"这一刻，主动看成"一头大象走到了我面前"（觉察，不逃避）
  看：点击身体部位图标（心念·通过身体）
  问：点击需求标签（安全感/被认可/掌控感/链接感）
      → 淡显一句活人话语言转换（3 秒消失）
      → 下方「这句话触动了我」轻触按钮（1 秒，不强制）

[晚间复盘] 2 分钟，可跳过
  滑动条（好难↔还好）+ 活人话 AI 总结（内在视角，不评判）
```

### Flow B · 深度工具（用户拉起，产品绝不主动推）

```
[四步照象] 暂停法化不开时，产品才淡出"这头有点大，拆一拆？" → 用户点了才进
  第1格 现象：点生活领域芯片（含"说不清"）
  第2格 追问：点"我为什么这么在意"芯片（含"说不清"）
  第3格 需求：点四需求芯片 → 第2→3 格自动翻译（AHA："你说怕不被认可，其实你需要被看见"）
  第4格 第一步：按需求弹 3 张活人话行动卡挑一张（可"先不定"）
  全程 ~60 秒，存 4 个 enum，零文本

[求助过滤网] 想向外抓取时，用户自己选"问之前，先照一眼"
  4 次点选（懂的人/问题类型/是否自行动过/换视角还是偷答案）
  → 活人话结论（10 秒自检，不是反思长文）

[手写逃生舱] 可选本地便签，默认折叠
  标注"只存在你手机上，我们看不到、也不上传"
```

**双场景粘合剂**：草原 / 大象。小大象（日常）和大大象（深度）长在同一片草原上，两种场景天然是一个产品。

## 3. 功能规格（P0–P3）

### P0 · MVP（验证核心价值）
- 大象暂停法完整闭环：停 / 看 / 问
- 晨间锚点（3 选 1 + 活人话淡显）
- 晚间复盘（滑动条 + 活人话总结）
- **活人话语言转换库**（按 4 需求标签各配台词；语气铁律：系统=平等陪伴者、共情在前、谦和非强制建议，禁禅意/诗意/**俯视诊断腔**）
- 「这句话触动了我」轻触按钮
- 本地存储（不联网也能用）

### P1 · 早晚态 + 数据 + 飞行模式
- 早晚界面切换（晨锚点 / 日脚印 / 夜放手）
- `phraseTouched` 字段落库
- **飞行模式物理断联**（原稿 MVP 必杀技、付费核心卖点）：进入暂停法流程时自动开系统专注模式、流程结束前屏蔽微信/邮件等外界通知——把"向内求"从虚拟概念变成**物理级向外断联**
- 基础草原气泡图（行为数据：横轴时间、纵轴需求标签、气泡=频率）

### P2 · 三层可视化 + 深度工具
- 「我的草原」页极简说明（100 字内，方案 B，点破能量/磁场大白话）
- **草原呈现 = 融合**（底层沿用《产品构想》抽象水墨气泡图：横轴时间、纵轴需求标签、气泡=频率，只显"需求密度"不显"负面情绪"；表层为卡通治愈草原场景，大象在草地上，大小/位置/表情编码需求频率与最近状态）
- **四步照象**（全点选，含"说不清"选项）
- **求助过滤网**（4 点选分流）
- 手写逃生舱（本地便签）
- 草原气象可视化（需求模式变化趋势 = 能量/磁场的隐喻表达）

### P3 · 长期镜像 + 同步
- AI 周报（含「你的能量和磁场」板块，大白话镜像，不评判）
- 草原变迁（3 个月需求模式演变，地形变迁图）
- 多端同步
- **惊喜里程碑**（如"你刚才那下是第 20 次面对大象——你其实已经不怕了"，自然蹦出，非强制成就）

## 4. 关键交互规范

- **语言转换淡显**：3 秒自动消失，非输入，仅展示。按需求标签给活人话。语气铁律：**系统=平等的陪伴者/镜子，不是上方评判者、不是教练指挥官**；共情在前（"这会儿心里乱，正常"），不替用户贴"你慌/你怕"标签；建议谦和可选（"不妨/也行/就好"），绝不"你应该/必须"；**禁止禅意/诗意/俯视诊断腔**。
- **「触动了我」**：1 秒轻触，不强制。点了 = 这条语言转换真正落进用户心里。
- **深度工具入口**：暂停法化不开 → 才淡出"这头有点大，拆一拆？"，用户点了才进。**产品永远不主动弹深度工具**。
- **每格"说不清"合法**：四步照象每一格都保留"说不清"，对应"模糊复杂、可能没觉察"的真实状态，绝不逼用户先想清楚才让用。
- **零缺陷暗示**：全程不出现"你有问题 / 你不完整 / 你需要修"。入口让"我很好"成为合法完整回答。

## 4.5 声景规范（Soundscape · David 2026-07-17 新增）

**核心立场**：声音是"点睛的单声"，不是背景音乐，更不是通知音。它服务于"停下来、静下来、向内看"，**绝不能喧闹、绝不能喧宾夺主**。

- **默认极轻、可整体静音**：全局一个静音开关；单声音量默认很低，环境嘈杂时听不清也不影响使用（视觉才是主通道）。
- **一声即止，留白优先**：只用短促单声（≤1.5 秒）+ 自然衰减长尾，绝不循环、绝不铺底噪、绝不连播。声音之后是安静，不是下一段声音。
- **只在关键锚点响，其余全静**：
  - **停**（息屏那一刻）：一声低频、温暖的"钵 / 磬"轻响，帮用户从外界抽离——听觉上的"大象来了"。
  - **触动了我**（轻触确认）：一声极轻的"涟漪 / 叶落"回应，像草原轻轻应了一下，不是"叮"的奖励音。
  - **晨间锚点 / 晚间复盘进入或收束**（P1）：可选一声更柔的过渡音，默认关。
- **绝不出现的声音（听觉版行业 guilt，从第一天禁止）**：通知提示音、成就 / 连胜音、弹窗"叮"、错误"嗡"、任何催促或奖励性音效。
- **触觉与听觉配合**：息屏时的轻震动（haptic）与钵响同刻，物理层与听觉层一起把人"接住"，但两者都克制。
- **哲学一致性**：声音属于环境支撑，绝不替用户下判断、不评价对错；它只是草原的自然回声。

## 5. 数据模型（只存枚举，绝不存文本）

> 完整定义见前期研究 v0.2 第五、六章。此处列核心结构。

```typescript
// 日常轻循环
MomentRecord {
  id, userId, phase, tag(NeedTag?), bodyPart?,
  phraseTouched: Boolean?,   // 「这句话触动了我」是否被轻触
  createdAt
}

// 四步照象（深度工具）
ElephantAnatomy {
  id, userId,
  domain: DomainTag,       // 工作/家里/某个人/钱/自己状态/说不清
  probe: ProbeTag,         // 怕失控/怕不被认可/怕让人失望/咽不下这口气/就是累了/说不清
  need: NeedTag,           // 被认可/安全感确定性/掌控感/链接感
  action: ActionTag?,      // 按需求弹出的活人话行动卡；"先不定"则空
  createdAt
}

// 求助过滤网（深度工具）
HelpFilter {
  id, userId,
  personKnows: Enum,      // 懂/陪吐槽/不确定
  questionType: Enum,     // 具体卡点/我该怎么活
  selfPrepared: Enum,     // 想过查过/还没
  intent: Enum,           // 换视角/偷答案
  createdAt
}
```

**隐私铁律**：以上全部只存枚举 / 整数，**绝不存任何用户输入的文本**。手写逃生舱内容仅留本地，不进入任何上述模型，也不上传。

## 6. 非功能需求

- **隐私优先**：枚举 + 本地优先，"我们不看"可验证（比竞品"数据本地"更强的信任状）。
- **零诱导**：不 push 消费、不制造愧疚、不用连胜/徽章施压。
- **性能**：冷启动 < 1 秒；单次核心交互 < 60 秒。
- **离线优先**：无网也能用核心循环，同步为可选项。
- **低负担**：点击远大于输入；定时件（晨/晚）均可跳过。

## 7. 度量与埋点

记录（**全部为枚举 / 计数，无文本**）：
- 各需求标签 / 领域 / 追问 / 行动枚举的选择分布
- `phraseTouched` 触达与比率
- 触发场景分布（日常暂停 / 深度照象 / 过滤网）
- 留存 cohort（周 / 月）

**不记录**：任何自由文本、任何用户输入的心理内容。

---

## 8. 范围外（明确不做）

- ❌ 不做冥想 / 呼吸计时器（那是竞品场，见象是镜子不是练习场）
- ❌ 不做写作日记 / CBT 长记录（写作负担是我们要解的痛点）
- ❌ 不做诊断 / 不替代心理咨询（镜子不是医生）
- ❌ 不做连胜 / 徽章 / 愧疚式提醒（这是行业流失根因）
- ❌ 不做诱导式课程转化（不学壹心理推课翻车）
