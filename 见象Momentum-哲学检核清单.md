# 见象（Momentum）· 实现哲学检核清单

> 用途：每次写代码 / 出原型 / 改交互前，逐条过一遍。任何一条不过，先停下对齐，不要"先写出来再说"。
> 来源：David 2026-07-16 重申的不可偏离核心脊柱 + 前期研究 v0.2。

## 一、核心脊柱（不可违背）

**1. 向内求本质定义（5 步因果链）**
```
觉察问题（不逃避）→ 追问本质（为什么）→ 理解需求（看见大象）
→ 主动行动（学习/求助/改变）→ 扩容心量（接纳无常）
```
任何功能都不得打断或替代这条链。

**2. 关键判据（最易漂移，重点盯）**
> **是否向内求，只看行为背后内心的想法——是被动等待，还是主动付出。**
- 做了动作（停/看/问）但内心仍在"等别人解决 / 等问题自己消失" = 被动等待，不是向内求。
- 真向内求 = 主动理解、主动付出、主动面对。
- ⚠️ 北极星"周主动触发率"必须度量"理解/主动"的质量，不能只度量"打开了"——否则会激励"点一下把感觉打发掉"的被动动作，与判据正相反。

**3. 三层改变路径（缺一不可，语言是桥）**
```
心念（内在觉察）→ 语言（自我对话/对外表达）→ 行为（实际动作）
```
- 光改心念不改语言 = 觉察到但嘴上还在抱怨
- 光改行为不改心念 = 做了动作但内心仍逃避
- **语言是心念和行为之间的桥梁**（见象相对竞品最独特的层）

**4. 对人生轨迹的影响（不直接宣讲，靠设计让用户自然体验到）**
- 面对挑战：心量扩大后，挑战不再是威胁而是课题
- 人际关系：语言改变后，关系从对抗变成协作
- 个人成长：行为从逃避变成主动后，成长不再靠运气而是靠积累
- 完型（Gestalt）：三层一致时，内在不再分裂——心念、语言、行为指向同一方向，人的内在完整性恢复

## 二、实现前逐条检核（8 问）

- [ ] **判据红线**：这个功能在帮用户"主动理解 / 主动付出"，还是只帮"把感觉打发掉"？
- [ ] **因果链**：它落在 5 步里的哪一步，且不破坏整条链？（觉察→追问→理解→行动→扩容）
- [ ] **三层承接**：心念 / 语言 / 行为三层是否都有承接，还是只动了一层？（尤其别漏"语言"这层）
- [ ] **零缺陷暗示**：有没有任何"你有问题 / 你不完整 / 你需要修"的味道？
- [ ] **入口包容**：入口是否让"我很好"也成为合法、完整的回答？
- [ ] **隐私铁律**：只存枚举、绝不存文本？本地优先、"我们不看"可验证？
- [ ] **无 guilt**：无连胜 / 徽章 / 愧疚式提醒？北极星奖励的是"主动理解"而非"被动打卡"？
- [ ] **隐喻不削弱**：草原 = 我们面对的整个世界、大象 = 任何让自己面对的东西？隐喻有没有被削弱或说教化？

## 三、语气铁律（语言层专属，v0.2 修正后）

系统角色 = **平等的陪伴者 / 镜子**，不是上方评判者、不是教练指挥官。
- 共情在前（"换谁都会…""谁都会有点…"的共通人性），不替用户贴"你慌 / 你怕"标签
- 建议谦和、可选、非强制（"不妨 / 也行 / 就好"），绝不用"你应该 / 必须"
- 活人话、简洁直接、好懂；**禁禅意 / 诗意 / 俯视诊断腔**
