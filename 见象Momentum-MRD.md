## 1. 市场背景与趋势

| 指标 | 数据 | 含义 |
|------|------|------|
| 中国正念/冥想市场（2025） | 约 65–160 亿元（不同口径） | 口径差异大但都指向"已成规模" |
| 中国用户基数 | 超 1.2 亿 | CAGR 25%+ |
| 全球冥想 APP（2025） | 约 22–40 亿美元 | 年增 20%+ |
| 情绪消费大盘（2025） | > 2 万亿元 | Z 世代为情绪价值溢价 30–50% |
| 人均年消费 | 215 元（2025E，2023 为 185） | 订阅制成主流，潮汐"基础免费+高级订阅"年营收破亿 |

**政策顺风**：《健康中国 2030》明确鼓励互联网心理健康服务。

**最关键的趋势信号**：市场正从"内容/课程"走向**轻量化仪式 + 即时急救**。
- 电子木鱼 DAU 500 万（95 后占 72%），本质是"用轻量仪式对抗焦虑"。
- 冥想类 APP 全球下载同比 +218%。
- 年轻人要的是"5 分钟情绪急救"，不是 45 分钟课程。

## 2. 目标用户与真实状态

**核心洞察（David）**：用户不是"带着问题来"，是"带着一团模糊复杂、说不清的状态来"——他自己可能都没觉察；有改变的念头，但一直没动，也不知道从哪开始。而且**没人承认"我有问题需要改变"**。

**两类状态（双场景，见前期研究第五章）**
- **日常**：小烦躁 / 小焦虑 / 想发火——高频、轻、随时冒出来。
- **大困扰**：问题像大象一样挡路——低频、重、需坐下来拆。

**用户画像（基于调研）**
- 主力：25–45 岁，一线 / 新一线白领中产，女性略多；下沉市场增速最快（+30% YoY）。
- 心理特征：**高功能人群**——能扛事但不被接住；有"自我探索"意愿但抵触"被当成病人"；**先看书 / 搜资料 / 做测评（62% / 47%），才考虑找人（仅 30% 找专家）**。
- 明确排斥：被诊断、被说教、被要求"坚持打卡"、被 push 消费。

## 3. 核心痛点（现有产品都没解决好，全部有调研支撑）

### A. 留存崩塌（冥想 / 正念类：潮汐、小睡眠、Calm、Headspace）
- **95% 用户 30 天内流失**；中位 15 日留存 3.9%、30 日 3.3%、DAU 4.0%（Baumel et al., JMIR 2019，93 款 app）。
- **根因 = "被动 app 陷阱"**：你不主动开它，它就永远沉默；凌晨 1 点头最乱时它不会找你。这是**时机问题，不是意志力问题**。
- **连胜 / 徽章制造愧疚感** → 驱赶用户（调研原话："skip 一天就被 app 骂"）。
- 名人语音 / guided meditation 反噬（whispery 声音让焦虑的人"想砸墙"）。
- $80/年难支撑（有免费呼吸计时器 + YouTube）。

### B. 写作负担（CBT / 自助类：驻念、思想日记 Clarity、意愈）
- 核心是"写"——写下灾难化念头 / 思维记录 / 日记。但**情绪最乱时最没力气写**，高功能人群"道理都懂，就是做不到"。
- 结构化流程重（意愈 8 步、Clarity 60+ 练习），完不成 = 又一项没做好的事。

### C. 病耻感与防御（心理咨询类：壹心理、简单心理、KnowYourself）
- "很多人羞于承认自己糟糕的情绪状态，并不能真正地接受心理咨询"（第一财经）。
- **诱导转化翻车**：用户做完测评就被推课，"平台应该多科普而不是急着转化"（用户原话）。
- 客单价高（400–600 元/次），心理门槛高。

### 见象的破法对应
- A → 不靠连胜，靠**生活触发 + 极简工具**
- B → **点击 > 输入**，不写
- C → **不诊断不评判**，"我很好"也是合法入口

## 4. 价值主张（差异化）

见象不卖"解决你的问题"，卖"**扩容你面对问题的心量**"。

> 一句话价值：见象是一面随时能打开的镜子——把你眼前那头大象照出来，不拔、不躲、不解决，就看着。看着看着，它没那么大了，你脚下的草原也宽了。

**定位金句（直接引用《产品构想》原稿）**
- "大脑的健身房，但不是让你举铁，而是教你深呼吸。"
- "不是心理咨询师，也不是待办清单，而是'沉默的提醒者'。"
- "它必须比刷短视频还快、还爽，才能真正替代旧习惯。"（产品构想对极简的最后一条硬要求）

**差异化三角**
1. **镜子不是诊断**：零缺陷暗示，不暗示"你有问题 / 你不完整"。入口让"我很好"成为合法、完整的回答。
2. **工具不是治疗**：5 秒，点击 > 输入，不写不聊不分析。治疗暗示你病了，工具不。
3. **三层改变 + 语言桥**：心念→语言→行为，语言是桥。把 CBT 认知重构做成"3 秒淡显 + 轻触"，不是写作练习。

## 5. 竞争分析

| 梯队 | 代表产品 | 强在哪 | 弱在哪 |
|------|---------|--------|--------|
| 冥想 / 正念内容 | 潮汐、小睡眠、冥想星球、Calm | 内容生态、品牌 | 留存崩塌、被动陷阱、愧疚式连胜 |
| 心理咨询平台 | 壹心理、简单心理、KnowYourself | 专业供给、循证 | 病耻感门槛、高客单、诱导转化翻车 |
| CBT / 自助 | 驻念、思想日记、意愈 | 循证、结构化 | 写作负担、流程重、情绪乱时写不出 |

**见象占据的空白带**：极简工具 + 不写 + 不诊断 + 语言层桥 + "扩容心量"哲学 + **物理级断联（飞行模式）**。目前**无一家占据**——竞品最多做到"提醒你深呼吸"，没有一家能在你点下去的瞬间**物理屏蔽外界通知**。

## 6. 为什么是现在

- **轻量化仪式趋势已验证**（电子木鱼证明"低承诺入口"可行）。
- **病耻感催生 self-serve 前置需求**（用户先自助 62%/47%，才考虑专家 30%）。
- **AI 成熟**可做"镜像"而非"生成内容"，且隐私技术（本地 / 枚举）能兑现"我们不看"的承诺。
- **政策 + 资本双顺风**。

## 7. 增长钩子（语言层口碑飞轮）

来自前期研究的增长逻辑（实现阶段再用，此处定方向）：
- 一次"啊哈"转换（"他凭什么"→"我需要被看见"）本身就是句能说出口的话，用户会当面朋友说。
- 可分享单元 = 那张"转换卡" / "草原气象"截图，relatable 不尴尬。
- 草原 / 大象词汇变成用户社交语言（"你那头大象是啥？"）。
- 「触动了我」按钮告诉产品哪些转换最共鸣 → 浮出最常被分享的几句 → 内容飞轮。
- **核心**：心念→行为是你自己的桥，语言层是你替产品拉新的桥。

## 8. 商业模式设想（对齐《产品构想》原稿）

> 以下结构与定价直接采用《产品构想》原稿设定（经 David 确认）：

- **免费版**：无限次"大象暂停法" + 最近 7 天数据看板。
- **付费订阅（年费制，象征性 ¥68/年）**：解锁"草原图谱"无限历史回溯 + 每周 AI 周报 + 自定义"需求标签"（如增加"我需要独处"）。
- **坚决不做**：广告、付费咨询引流、社交排行榜——违背"向内求"内核（原稿"三不做"）。
- 订阅制成主流，但必须避开"$80 太贵" + "诱导转化"两个坑；**隐私即信任**："只存枚举不存文本"是比竞品"数据本地"更强的信任状，直接转化付费意愿。
- **严禁诱导转化**（不学壹心理推课翻车）。

## 9. 风险与关键假设

- **假设 1**：用户会为"扩容心量"而非"解决具体问题"持续回来 → 需用留存数据验证（目标：显著优于行业 3–4% DAU）。
- **假设 2**：不靠连胜 / 提醒也能养成习惯 → 靠"生活触发反射" + 轻量晨晚锚点引导。
- **假设 3**：极简不损深度 → 四步照象 / 过滤网证明"点选也能照出深度"。
- **风险**：概念太抽象难上手 → 首屏 AHA 必须一秒戳中"那团模糊"。
